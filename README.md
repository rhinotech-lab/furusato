# AI Studio アプリの実行とデプロイ

このリポジトリには、アプリをローカルで実行するために必要なものがすべて含まれています。

AI Studio 上のアプリ: https://ai.studio/apps/drive/1z4lf04q4FJqnINDcp6GJ5po_cmp6OgkD

## ローカルで実行

**前提条件:** Node.js

1. 依存関係をインストール:
   `npm install`
2. Gemini API キーを `.env.local` に設定:
   `GEMINI_API_KEY` を設定してください
3. アプリを起動:
   `npm run dev`
